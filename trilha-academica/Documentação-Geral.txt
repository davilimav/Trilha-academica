## 2. Visão Geral da Arquitetura

O sistema utiliza arquitetura cliente-servidor em 3 camadas:

### 2.1 Frontend

* HTML5
* CSS3 (variáveis globais)
* JavaScript Vanilla
* Chart.js
* html2pdf.js

---

### 2.2 Backend (Node.js)

* Node.js 18+
* Express.js
* JWT Authentication
* bcryptjs
* multer
* nodemailer
* axios
* mysql2

---

### 2.3 API Python (Machine Learning)

* Python 3.11+
* Flask
* Flask-CORS
* NumPy
* scikit-learn
* gunicorn

---

### 2.4 Banco de Dados

* MySQL 8.0
* Clever Cloud (produção)
* PostgreSQL (futuro)

## 7. API de Recomendação (Python)

### Endpoints

* GET /health
* GET /ping
* POST /recomendar
* POST /questionario/recomendar

---

### Exemplo Request

```
{
  "programacao_nivel": 2,
  "dados_nivel": 1,
  "design_nivel": 1,
  "seguranca_nivel": 0,
  "marketing_nivel": 0
}
```

---

### Fluxo

1. Frontend envia questionário
2. Backend repassa para API Python
3. Calcula nível médio
4. Aplica similaridade
5. Retorna recomendações

---

## 8. Autenticação

### Fluxo

1. Login do usuário
2. Backend valida credenciais
3. Gera JWT
4. Armazena no localStorage
5. Middleware valida rotas

---

### Estrutura JWT

```
{
  "id": 1,
  "name": "Usuario",
  "email": "email@email.com",
  "is_admin": 0
}
```

## 11. Problemas Comuns

### MySQL não conecta

* Verificar .env
* Banco ativo
* Credenciais

---

### API Python falha

* Verificar URL
* Logs do Render

---

### JWT inválido

* Token expirado
* JWT_SECRET incorreto

---

### Upload falha

* Permissão de pasta
* limite 5MB
